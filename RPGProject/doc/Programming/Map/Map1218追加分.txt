定数 :
const int SCREEN_XSIZE = 20
const int SCREEN_YSIZE = 15

フィールド :
int scX   //スクリーンの現在x位置
int scY   //スクリーンの現在y位置

Update()
マイキャラの処理終了後、
scX = マイキャラのx位置-SCREEN_XSIZE/2	(ただし、0未満になるならscX=0 (xSize-SCREEN_XSIZE)以上になるならscX=xSize-SCREEN_XSIZE
scY = マイキャラのy位置-SCREEN_YSIZE/2  (上記と同様、0未満とySize-SCREEN_YSIZEで判定)


Draw()
描画するマップチップの範囲を、
scX ~ scX+SCREEN_XSIZE
scY ~ scY+SCREEN_YSIZE
とする。
具体的には、for文で定義しているx,yが、SCREEN_XSIZE,SCREEN_YSIZEとなるまで繰り返し、
drawTipにアクセスしているx要素にscXを、y要素にscYを足せば良い


GetScreenX() GetScreenY()
scX,scYを返す
