名前空間 : RPGProject.GamePlay.Party
クラス名 : Player
概要 : プレイヤーそのものを表します。
メンバ :
string name : 名前
PlayerStatus status : ステータス
int Lv : レベル
int HP : 現在HP
int TP : 現在TP

コンストラクタ :
nameとPlayerStatus、HP/TPを引数で受け取ってフィールドを初期化。

GetHP GetTP :
HP/TPのゲッタ

SetHP SetTP :
HP/TPのセッタ
0以上MaxHP/TP以下になるように、引数をチェックする。
0未満の値なら、0を代入
MaxHP/TPを超える値なら、MaxHP/TPを代入

GetStatus :
statusのゲッタ

GetLv : 
Lvのゲッタ
