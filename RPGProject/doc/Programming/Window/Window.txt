名前空間 : RPGProject.GamePlay.Window
クラス名 : Window
概要 : なにもない空白のウィンドウを表します。
メンバ :
int px : 左上座標
int py : 右上座標
int width : 幅
int height : 高さ
bool broken : 破棄されているかどうか

Window(int arg_px, int arg_py, int arg_width, int arg_height) : コンストラクタ
処理 :
メンバに引数を代入
brokenにfalseを代入

virtual void Update() : 更新メソッド
処理 : 
Inputの内容を受け取って、INPUT_DECIDEの入力が1以上ならbrokenをtrueにする

virtual void Draw() : 描画メソッド
処理 :
四角形を描画する。（左上座標(px,py) width×height）

bool isBroken() : 破棄されているかどうか
brokenの値をそのまま帰す

void Break() : 破棄する
brokenの値をtrueにする
